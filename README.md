# Copilot SmartCards

This repository contains the source code for the application that scans and executes the NFC chips on the prompt cards that we'll use at the SETT Mechelen 2025 event.

# Building and running

To build and run this program, open it in Visual Studio.

Make sure that the `Package` project is set as startup project, and that you select an architecture other than `Any CPU`.

Then just hit "F5" to build and run the program
